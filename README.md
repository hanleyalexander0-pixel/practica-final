# practica-final
Descripción

En este programa lo que hice fue crear un sistema sencillo para registrar estudiantes con sus calificaciones. El usuario puede ingresar hasta 20 estudiantes, escribir su nombre y luego ingresar 4 notas por cada uno.

Después de ingresar los datos, el programa calcula automáticamente el promedio de cada estudiante, el promedio general del grupo, la nota más alta, la más baja y también muestra cuántos aprobaron y cuántos reprobaron.

También me aseguré de validar que todas las notas estén entre 0 y 100 para evitar errores.

# como-funcionan-los-arreglos
En mi programa usé arreglos para guardar los datos de los estudiantes. Utilicé un arreglo llamado nombres[] para guardar los nombres y otro llamado promedios[] para guardar el promedio de cada estudiante.

Cada posición del arreglo representa a un estudiante. Por ejemplo, en la posición 0 se guarda el primer estudiante, en la 1 el segundo, y así sucesivamente.

Usé un ciclo for para recorrer los arreglos, ingresar los datos y luego mostrar los resultados. Esto hace que el programa sea más ordenado y fácil de manejar.
